# Explainable Hybrid Ensemble Framework for Multiclass DDoS Attack Detection

## Overview

This project presents an Explainable Hybrid Ensemble Machine Learning Framework for Multiclass DDoS Attack Detection using advanced tree-based machine learning models and explainable AI techniques.

The system is designed to classify multiple categories of network traffic including:

* Benign
* LDAP
* MSSQL
* NetBIOS
* Syn Flood
* UDP Flood
* UDPLag

The framework combines the power of:

* Random Forest
* XGBoost
* LightGBM
* SHAP Explainability

for intelligent and scalable intrusion detection.
